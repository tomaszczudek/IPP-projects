# IPP Projects

School projects created as part of the IPP course.

## Description
This repository contains two projects focused on processing and interpreting the SOL25 language.  
The first project is a parser written in Python, and the second project is an interpreter implemented in PHP.

## Projects

### Project 1
Parser for the SOL25 language.  
The project performs lexical and syntactic analysis, basic semantic checks, and generates an XML representation of the input program.

### Project 2
Interpreter for the XML representation of SOL25.  
The project processes the XML input, builds an internal representation, and executes the program using an object-oriented design.

## Technologies
- Python
- PHP
- XML
- DOM / ElementTree
- Makefile

## Run

### Project 1
```bash
python3 proj1/project1/parse.py < input.txt
```

### Project 2
```bash
php proj2/ipp-core/interpret.php
```
